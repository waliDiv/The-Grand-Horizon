## 🔍 Overview

The Grand Horizon is a complete, production-ready hotel website template designed for luxury accommodations. It provides an elegant online presence with intuitive navigation, engaging visuals, and conversion-focused elements like booking modals and clear calls-to-action.

## ✨ Features

### Design & UX
- **Luxury Aesthetic**: Clean, modern design with elegant typography (Playfair Display & Roboto)
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Sticky Navigation**: Persistent navbar with smooth scrolling
- **Hero Carousel**: Full-width image slider with text overlays
- **Hover Animations**: Interactive cards, buttons, and images
- **Semantic HTML**: SEO-friendly structure with proper tags

### Functionality
- **Interactive Elements**: Booking modal popup on all pages
- **Smooth Scrolling**: Enhanced navigation experience
- **Contact Form**: Working form with validation (demo)
- **Google Maps Integration**: Embedded map on contact page
- **Social Media Links**: Connected footer icons

### SEO & Accessibility
- Meta descriptions and keywords for each page
- Open Graph tags for social sharing
- Alt attributes for all images
- High color contrast for readability
- ARIA-compatible Bootstrap components

## 🛠 Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with animations
- **Bootstrap 5** - Responsive framework
- **JavaScript (ES6)** - Interactive features
- **Google Fonts** - Playfair Display & Roboto
- **Bootstrap Icons** - Vector icons and symbols
